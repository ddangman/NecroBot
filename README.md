 - [PTC Login / Google]
 - [Get Map Objects and Inventory]
 - [Search for forts/spawns]
 - [Farm forts]
 - [Farm all Monster in neighbourhood]
 - [Throw Berries/use best ball]
 - [Transfers duplicate Monsters]
 - [Evolve / Upgrade all Monster]
